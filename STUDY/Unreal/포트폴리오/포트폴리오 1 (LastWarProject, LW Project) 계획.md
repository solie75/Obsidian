
# 플레이어

원거리 무리를 사용하여 다가오는 적들을 죽이고 보스몹을 처리한다.
# 기본 스테이지

- 이동 : 플레이어의 횡 이동. 공격 방향 고정 (w 방향)
- 오브젝트 : 기믹, 일반 몬스터,  네임드 몬스터

# 보스 스테이지

- 이동 : 플레이어의 전방위 이동. 공격 방향 실시간 변경(마우스가 위치한 방향)
- 오브젝트 : 보스 몬스터
