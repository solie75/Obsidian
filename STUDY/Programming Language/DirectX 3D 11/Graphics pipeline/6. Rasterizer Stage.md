rasterization stage 는 3D 그래픽의 실시간 diaplaying을 목적으로 vector 정보(shapes 또는 primitives 로 구성된)를 raster image (픽셀로 구성된) 로 변환한다. 

<span style="color: yellow">rasterization 중에 각 primitive 는 pixel로 변환되며 각 primitive 의 정점당 값을 보간한다.</span>  rasterization 은 view frustum 에 대한 정점을 클리핑하는 것, 원근법(perspective)을 제공하기 위해 z 로 나누는 것을 수행하는 것, 2D viewport 레 대한 primitives 의 매핑(mapping)하는 것 그리고 pixel shader 를 작동시킬 방법 을 포함한다. 



래스터화 중에 각 프리미티브는 픽셀로 변환되며 각 프리미티브에서 정점당 값을 보간합니다. 래스터화에는 보기 절두체에 정점 자르기, 원근을 제공하기 위해 z로 나누기 수행, 기본 요소를 2D 뷰포트에 매핑, 픽셀 셰이더 호출 방법 결정이 포함됩니다. 픽셀 셰이더를 사용하는 것은 선택 사항이지만 래스터라이저 단계는 항상 클리핑, 점을 균일한 공간으로 변환하기 위한 원근 분할을 수행하고 정점을 뷰포트에 매핑합니다.
