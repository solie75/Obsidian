
## move semantic

주로 rvalue 참조와 함께 사용된다. rvalue 는 이름이 없는 임시 객체 를 가리키는 표현식이다. rvalue 는 '&&' 연산자를 사용하여 선언된다.

이동 시맨틱스를 구현하기 위해서 c++11 부터 이동 생성자(move constructor)와 이동 할당 연산자(move assignment operator)가 도입되었다. 

이동 시맨틱스는 보통 다음의 경우에 사용된다.
1. 임시 객체 생성 : 함수가 임시 객체를 반환할 때, 이동 시맨틱스를 사용하여 복사를 피할 수 있다.
2. 동적으로 할당된 자원 관리 : 동적으로 할당된 자원을 가진 객체를 다른 객체로 이동하여 자원 누수를 방지할 수 있다.
3. 컨테이너의 요소 이동 : 벡터나 리스트 등의 컨테이너에서 요소를 이동할 때 복사보다 효율적으로 메모리를 관리할 수 있다.
