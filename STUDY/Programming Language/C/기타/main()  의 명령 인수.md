main 함수의 매개변수는 보통 아무것도 사용하지 않지만 다음의 형태도 가능하다.

```c++
int main(int argc, char* argv[])
```
1. int argc - 메인함수에 전달되는 정보의 갯수
2. char* argv[] - 메인함수에 전달되는 실질적인 정보. 문자열의 배열로 첫번째 문자열은 프로그램의 실행결로로 항상 고정되어 있다.
